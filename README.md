# QA Testing Project – REST API Testing

## Overview
Manual and automated API testing for a dummy REST API using Postman.

## Tools Used
- Postman (Collection + Report)
- Newman (for CLI execution)

## Tests Covered
- GET Users
- POST Create User
- PUT Update User
- DELETE User
